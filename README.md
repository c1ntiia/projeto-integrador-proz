# Aços e Sustentabilidade 🌍

Este site foi desenvolvido como parte de um Projeto Integrador do curso **Proz - Liga Steam**.
Seu objetivo é acadêmico e educativo, voltado para a aplicação prática dos conteúdos estudados em aula. 
Este site busca conscientizar sobre o papel do aço na **economia circular**, destacando sua reciclabilidade e importância para um futuro mais sustentável.


## Sobre o Projeto
O aço é um dos materiais mais reciclados do mundo.  
O site apresenta conteúdos educativos sobre:
- O **processo de produção do aço** e suas etapas industriais
- A importância do aço para a **sustentabilidade**
- Exemplos de reaproveitamento metálico e redução de impactos ambientais
- Ações práticas e iniciativas voltadas para a preservação ambiental


## Páginas do Site
- **Etapas do Processo de Produção do Aço**  
  Diagrama ilustrativo mostrando desde o minério até o produto laminado.  

- **Aços e Sustentabilidade**  
  Explicação sobre como diferentes tipos de aço (carbono, inoxidável, galvanizado) contribuem para a economia circular e para práticas sustentáveis.  

- **Ações**  
  Página dedicada às iniciativas e práticas que podem ser adotadas para reduzir impactos ambientais, incentivar a reciclagem e promover hábitos sustentáveis no dia a dia.  


## Objetivos
- Incentivar a reciclagem e reutilização do aço  
- Mostrar como a indústria pode reduzir resíduos e emissões  
- Divulgar conhecimento acessível sobre sustentabilidade e inovação  
- Inspirar ações práticas que cada pessoa pode adotar para contribuir com o meio ambiente
